## What is Wynter Growth OS?

A **no-code / low-code multi-agent system** in **Gumloop** that acts as a complete Growth Operating System for B2B SaaS.

### Core Capabilities
- **Research Agent**: Pulls fresh trends, competitor messaging, and B2B insights using Firecrawl + Browser Use
- **Idea Machine Agent**: Generates 20–40 creative, high-attention marketing ideas with virality/attention scoring
- **Content Architect**: Produces full assets (LinkedIn threads, carousels, email sequences, blog drafts, landing page copy)
- **GEO Optimizer**: Rewrites everything for Generative Engine Optimization (new discovery layer)
- **Distributor Agent**: Preps posts + can send test outreach via AgentMail
- **Memory & Observability**: Logs everything to Supabase + traces via Langfuse

**Built with tools from my stack**:
- Gumloop (main orchestration + Gummie AI builder)
- Firecrawl (web research)
- Supabase (backend + storage)
- AgentMail (distribution)
- Langfuse (observability)
- Infisical (secrets)
- Tavus / sync.AI (optional video explainers)
